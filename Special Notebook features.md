## keyboard shortcuts

### Command mode 
- X cut selected cells
- C copy
- V paste below
- Shift V paste above 
- DD delete slected cell
- A insert cell above
- B insert below
- Ctrl Enter run selected cell
- Shift Enter Run sleceted cell below
- Shift M merge cells
- Shift L toggle line numbers
